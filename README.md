# Sistema Core Orchestrator

Bem-vindo ao node central do **Sistema**, uma plataforma SaaS de orquestração corporativa B2B de alta performance, focada em saúde, seguros e gestão financeira.

## 🚀 Tecnologias Core

- **Framework:** Next.js 15 (App Router)
- **Interface:** ShadCN UI + Tailwind CSS (Paleta Profissional: Azul Marinho #2966A3 e Accent Verde Água #2AD1D1)
- **Inteligência Artificial:** Genkit (Orquestradora Assistente com Pesquisa Completa por Ramo)
- **Backend:** Firebase (Authentication & Firestore Real-time)
- **Orquestração:** Conectores RPA e APIs para integração com sistemas legados (SAP, Folha de Pagamento)

## 🔑 Pilares Operacionais

1.  **Dashboard de Orquestração:** Monitoramento em tempo real de SLA e protocolos de RH.
2.  **Motor de Subscrição IA:** Avaliação de risco dinâmica e recomendações de planos.
3.  **Intelligent Claims:** Validação de documentos e detecção de fraude via visão computacional.
4.  **Security Hub:** Governança de dados sanitizados em total conformidade com a LGPD/GDPR.
5.  **Omnichannel Hub:** Chat de equipe (Grupo/Individual) e videoconferência segura.

## 🧠 Assistente: Orquestradora Neural SYS

A Assistente é o motor de inteligência do sistema, capaz de realizar "Pesquisas Completas" e fornecer análises técnicas profundas nos ramos de:
- **Seguros:** Análise de apólices e sinistralidade.
- **Imóveis:** Gestão de VGV e yield.
- **Financeiro:** Amortização e spread.
- **Saúde:** Gestão de crônicos e Rol da ANS.
